### What This Repository Contains

This monorepo contains the complete LitLabs AI platform:

1. **Web Application** (`/app`, `/components`, `/lib`)
   - Next.js 16+ web application
   - React 19.2.1 frontend
   - TypeScript strict mode
   - Tailwind CSS 4 styling

2. **Android Application** (`/android-app`)
   - Native Android app
   - Flutter module integration
   - Google Play Store ready

3. **API & Backend** (`/app/api`)
   - 37+ API endpoints
   - Firebase integration
   - Stripe payments
   - AI services

4. **Documentation** (Root `.md` files)
   - Comprehensive guides
   - Security documentation
   - Deployment instructions
   - Setup checklists

### Cloning and Contributing

To work with this repository:

```bash

git clone https://github.com/LiTree89/Labs-Ai.git

# Navigate to the directory
cd Labs-Ai

# Install dependencies
npm install

# Start development
npm run dev
```
