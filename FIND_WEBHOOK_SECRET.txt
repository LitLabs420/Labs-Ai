## Step-by-Step Guide

### 1. Go to Stripe Dashboard
```
https://dashboard.stripe.com
```

### 2. Click "Developers" (Bottom Left)
- Look at the left sidebar
- At the very bottom, you'll see "Developers"
- Click it

### 3. You'll see two options:
- API Keys
- **Webhooks** ‚Üê Click this one

### 4. Find or Create Your Webhook

**If you see a webhook listed:**
- Look for URL ending in `/handleStripeWebhook`
- Click on it
- Scroll down to find "Signing secret"
- You'll see something like: `whsec_test_...` or `whsec_live_...`
- Click "Reveal" to see the full secret
- Copy it

**If you DON'T see a webhook:**
1. Click **+ Add an endpoint** button
2. In "Endpoint URL" field, paste:
   ```
   https://us-central1-studio-4627045237-a2fe9.cloudfunctions.net/handleStripeWebhook
   ```
3. Under "Events to send", click **+ Select events**
4. Search for and select:
   - `checkout.session.completed`
   - `invoice.payment_failed`
   - `customer.subscription.updated`
   - `customer.subscription.deleted`
5. Click **Add endpoint**
6. Scroll down to "Signing secret"
7. Click **Reveal**
8. Copy the secret (starts with `whsec_`)

---

## What It Looks Like

The signing secret will look like:
```
whsec_live_51SYJoC440X4TKc4a1234567890...
```

---

## Once You Have It

Run this command:
```powershell
cd "C:\Users\dying\public"

firebase functions:config:set stripe.webhook_secret="whsec_live_YOUR_SECRET_HERE"

firebase deploy --only functions
```

Replace `whsec_live_YOUR_SECRET_HERE` with your actual webhook secret.
